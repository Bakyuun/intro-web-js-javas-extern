# Lab: Vinculando un archivos JavaScript externos

## Objetivos
- Agrega las etiquetas de apertura y cierre `<script><script>`
- Utiliza el atributo `src=""` para vincular un documento `.js`
- Utilizar las propiedades `top` y `left` para posicionar elementos
- Crear un variable de tipo constante(`const`)
- Utilizar document.querySelector() para llamar a un elemento mediante el DOM
- Utiliza el método .addEventListenner() para mandar a llamar una función

## Introducción 
Hemos aprendido que el atributo `src` que utilizaremos para vincular un archivo JavaScript es exactamente el mismo, pero ahora su valor especifica la ubicación del archivo JavaScript. 


## Funcionalidad del Código
En este lab crearas un documento javascript con la extensión `.js`
